# 代码块
提供各种组件，以代码块的形式编写。

### 项目结构

```$xslt
|--code-blocks::根项目, 不写任何代码,仅仅整合所有子项目
             |--code-blocks-common::通用模块
             |--code-blocks-spring::基于spring通用模块
             |--code-blocks-springboot::基于springboot通用模块
```
